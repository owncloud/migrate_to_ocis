# Contributing

Thank you for your interest in contributing to this project!

Please read the full contributing guidelines at:
**<https://owncloud.com/contribute/>**

For development setup, build and test commands, coding standards, and the
pull request process, see [`AGENTS.md`](AGENTS.md) and the [`Makefile`](Makefile)
in this repository.

## Developer Certificate of Origin

This project is licensed under the [Apache License 2.0](LICENSE), and all
contributions are accepted under those terms. We use the
[Developer Certificate of Origin](https://developercertificate.org/) (DCO)
instead of a Contributor License Agreement: by signing off on your commits you
certify that you wrote the contribution, or otherwise have the right to submit
it under the project's license.

Add a `Signed-off-by` line to every commit:

```
git commit -s
```

This appends a trailer using your `user.name` and `user.email`, for example:

```
Signed-off-by: Jane Developer <jane@example.com>
```

Pull requests are checked by the DCO bot and cannot be merged until every commit
is signed off. If you forget, amend or rebase your commits to add the sign-off
before requesting a merge.
